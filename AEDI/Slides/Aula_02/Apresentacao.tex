\section{Por que aprender a programar?}

\begin{frame}
  \frametitle{Por que aprender a programar?}
\begin{itemize}
\item Neste curso vocês aprenderão a básico para se criar programas para computador.
\item Exemplos de programas: Firefox, Word, MatLab, Call of Duty, OBS.
\item Programar é uma atividade básica de um cientista ou engenheiro.
\end{itemize}
\end{frame}		

\begin{frame}
  \frametitle{Por que aprender a programar?}
  \begin{itemize}
  \item Como engenheiro voc\^e deverá ser capaz de automatizar algum processo.
    \begin{block}{}
    Você poderá criar programas para gerenciar e automatizar algum e processo que hoje é manual.
    \end{block}
  \end{itemize}
\end{frame}		

\begin{frame}
  \frametitle{Por que aprender a programar?}
\begin{itemize}
\item Como engenheiro você deverá ser capaz de desenvolver novas ferramentas ou protótipos.
\begin{block}{}
Para criar ferramentas/protótipos você deverá fazer simula\c{c}ões computacionais para fazer testes preliminares.
\end{block}
\end{itemize}
\end{frame}

\begin{frame}
  \frametitle{Por que aprender a programar?}
Aprendendo a programar, você poderá:
\begin{itemize}
\item enxergar situa\c{c}\~oes onde uma solu\c{c}ão computacional pode trazer benefícios.
\item propor uma hipótese e testá-la computacionalmente.
\item resolver sistemas de equa\c{c}ões complexos que não necessariamente podem ser resolvidos por softwares padrões (como MatLab).
\begin{block}{}
Mesmo que você não programe a solu\c{c}ão você poderá propô-la e será capaz de interagir e com os responsáveis pela implementa\c{c}ão.
\end{block}
\end{itemize}
\end{frame}		

\begin{frame}
  \frametitle{Por que aprender a programar?}
\begin{block}{}Em vários casos onde os sistemas podem ser modelados matematicamente, são criados programas que fazem a simula\c{c}ões do 
sistema para checagem de uma hipótese.
\end{block}
\begin{block}{}
Vocês deverão implementar seus próprios resolvedores.
\end{block}
\end{frame}
