\begin{frame}[fragile]
  \frametitle{Introdução}
\begin{itemize}
 \item Os comandos de repetição são um recurso que permite que um certo trecho de código de um programa seja repetido um certo número de vezes.
 \item Na linguagem C, existem três comandos de repetição:
 \begin{enumerate}
  \item {\bf while}
  \item {\bf do-while}
  \item {\bf for}
 \end{enumerate}
\end{itemize}
\end{frame}

\section{Comando While}

\begin{frame}
  \frametitle{Comando While}
  O comando {\bf while} permite que um certo trecho de programa seja executado {\bf ENQUANTO} uma certa condição for {\bf verdadeira}.
\begin{figure}[!ht]
  \centering
  \includegraphics[width=150pt]{imgs/fluxograma_while.png}
\end{figure}
\end{frame}


\begin{frame}[fragile]
  \frametitle{Comando While}
A forma do comando {\bf while} na linguagem C é a seguinte:
\begin{lstlisting}
while ( condicao ) {
  comando1; // Comandos a serem repetidos
  comando2; // Comandos a serem repetidos
}
comando3; // Comando executado apos o while
\end{lstlisting}
\end{frame}

\begin{frame}[fragile]
  \frametitle{Comando While}
 O seu funcionamento é o seguinte:
\begin{enumerate}
 \item Testa a condição;
 \item Se a condição for {\bf verdadeira}, então executa cada um dos comandos do bloco subordinado ao comando {\bf while}.
 \item Ao executar o último comando do bloco {\bf while}, volta ao passo 1.
 \item Se a condição for {\bf falsa}:
 \begin{itemize}
  \item pula todos os comandos do bloco subordinado ao comando {\bf while},
  \item passa a executar os comandos após o bloco do comando {\bf while}.
 \end{itemize}
\end{enumerate}
\end{frame}
